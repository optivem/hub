**Milestone 1: Positive API Tests**

- [ ] At least one positive E2E test that creates a new entity
- [ ] At least one positive E2E test that updates an existing entity
- [ ] At least one positive E2E test that retrieves an existing entity


**Milestone 2: Negative API Tests**

- [ ] At least one negative E2E test with a null or empty input
- [ ] At least one negative E2E test with an invalid type input
- [ ] At least one negative E2E test with an invalid range input


**Milestone 3: Positive UI Tests**

- [ ] At least one positive UI E2E test that creates a new entity


**Milestone 4: Negative UI Tests**

- [ ] At least one negative UI E2E test


**Milestone 5: Parameterized Tests**

- [ ] At least one non-parameterized test
- [ ] At least one parameterized test


**Milestone 6: External System Limitations**

- [ ] At least one test where you can assert an exact output value because you control the external system
- [ ] At least one test where you cannot assert an exact output because you cannot control the external system
- [ ] At least one test where you cannot assert an exact output because of an uncontrollable factor (e.g. clock)
